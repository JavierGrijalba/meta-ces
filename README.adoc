Yocto Warrior 2.7
-----------------

Description
-----------

Official https://www.christ-es.de[Christ Electronic Systems GmbH] BSP Yocto Project layer.

Supported distros & images
--------------------------

Currently only ces-fb is supported with ces-console-image and ces-qt-demoimage.
